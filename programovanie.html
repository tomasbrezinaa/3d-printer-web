Programovanie

Programovanie je proces návrhu algoritmu, písanie, testovanie a ladenie zdrojového kódu počítačového programu.
Návrh algoritmu obvykle určuje všeobecný postup riešenia danej úlohy. Zdrojový kód programu je zápisom algoritmu v zvolenom programovacom jazyku.
Front-end je to informačná časť stránky ku ktorej má koncový uživateľ prístup
Back-end skupina informácii, ktorá sa nezobrazuje na web stránke iba ju riadi.

HTML HyperText Markup Language je štandardný značkovací jazyk pre dokumenty určené na zobrazenie vo webovom prehliadači.
Väčšinou sa k nemu pridávajú kaskádové štýly (CSS) a skriptovacie jazyky, ako je JavaScript.
Webové prehliadače prijímajú dokumenty HTML z webového servera alebo z lokálneho ukladacieho priestoru a vykresľujú dokumenty na multimediálne webové stránky. HTML prvky sú stavebnými kameňmi webových stránok.
HTML patrí medzi front-end programovacie jazyky.

CSS (Cascading Style Sheets) CSS tvorí dizajn web stránky, umožňuje oddelenie obsahu, vrátane rozloženia, farieb a fontov.
Dokáže zlepšiť dostupnosť obsahu, umožnuje viacerým webovým stránkam zdieľať formátovanie v samostatnom súbore .css, čo znižuje zložitosť. CSS patrí medzi front-end programovacie jazyky CSS je popri HTML a JavaScript základnou technológiou World Wide Web (www).

JavaScript JS, je programovací jazyk, ktorý patrí do skupiny základných technológií World Wide Web(www).
Javascript je objektovo orientovaný programovací jazyk.
Najznámejšie knižnice Javascriptu sú jQuery React.js Má aplikačné programovacie rozhrania (API) na prácu s textom, dátumami, regulárnymi výrazmi, štandardnými dátovými štruktúrami a objektovým modelom dokumentu
JavaScript je front-end aj back-end programovací jazyk.

Prvý obrázok : môžeme vidieť web stránku (jednoduché počítadlo) vytvorenú iba pomocou programovacieho jazyka HTML. Daná webstránka je nefunkčná a pre návštevníka nezaujimavá
Druhý obrázok : je tá ista web stránka ale je v nej použitý aj programovací jazyk CSS ktorý danú webstránku graficky upraví, ale web stránka (počítadlo) je stále nefunkčná
Tretí obrázok : vidíme stránku v ktorej je už použitý aj programovací jazyk JavaScript a počítadlo funguje , túto funkčnosť sme dosiahli práve vďaka spomínanemu JavaScriptu.
